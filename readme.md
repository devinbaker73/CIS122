### <div align="left"> Project Overview
<div align="left"> This projects was formed with the goal of creating a robot that can follow a line using sensor input and motor output. The robot design is based of a car, which also mirrors the real world application of this technology in self-driving cars like Tesla.<br/>
  
### <div align="left"> Materials:<br/>
Chassis: Cardboard<br/>
Brain: Arduino x 1<br/>
IR Sensor:  IR Infrared Obstacle Avoidance Sensor x 2<br/>
Motors: TT DC Gearbox Motor 3-6V Gear Motor x 4<br/>
Wheels: Simple casters x 4<br/>
